Описание проекта:
Данный проект представляет собой консольное приложение, разработанное на языке C#, предназначенное для хранения и управления списком пользовательских задач.

Назначение программы:
Программа позволяет просматривать список дел, добавлять новые задачи и удобно работать с ними через консольный интерфейс.

Версии программы:

Версия 1.0:

- Вывод заранее созданного списка задач
- Простая структура программы
- Базовый функционал просмотра данных

Версия 2.0:

- Добавлено меню управления
- Реализовано добавление новых задач
- Улучшен интерфейс программы
- Добавлена обработка ошибок ввода
- Цветовое оформление и нумерация списка

Функциональные возможности:

- Просмотр списка дел
- Добавление новых задач
- Меню выбора действий
- Выход из программы
- Сообщения об ошибках и успешных действиях

Используемые технологии:

- Язык программирования: C#
- Среда разработки: Microsoft Visual Studio
- Система контроля версий: Git
- Платформа хранения репозитория: GitHub

Инструкция по запуску:

1. Открыть проект в Visual Studio
2. Запустить программу (F5 или Ctrl + F5)
3. Выбрать нужный пункт меню
